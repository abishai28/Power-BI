# Power BI Dashboards

⚽ Premier League Team Statistics Dashboard

An interactive Power BI dashboard built to analyze player and team performance in the Premier League. The dashboard provides a comprehensive overview of key football statistics, enabling users to explore player contributions, disciplinary records, and team-specific insights through dynamic filters and visualizations.

🎯Project Objective: Develop an interactive Power BI dashboard that transforms raw Premier League player statistics into meaningful visual insights, allowing users to analyse player performance, disciplinary records, and team-level metrics through dynamic filtering and intuitive visualisations.

📊 Dashboard Features
Interactive Team Filter to analyse statistics for individual Premier League clubs.
Goal Scoring Analysis displaying the league's top goal scorers.
Yellow Card Distribution to identify the most booked players.
Red Card Distribution highlighting disciplinary trends.
Player Performance Comparison between appearances and goals scored.
Player Details Table containing jersey numbers, player names, and appearances.
Summary KPI Cards displaying:
⚽ Total Goals
🟨 Total Yellow Cards
🟥 Total Red Cards
✨ Key Highlights
Interactive filtering using slicers
Dynamic Power BI visualizations
Clean and modern dashboard layout
KPI cards for quick performance summaries
Comparative player analysis
Drill-down into team-specific statistics
Sports analytics using real-world football data

🛠️ Tools & Technologies

Kaggle Dataset
MS Excel
Claude(DATA cleaning)

📈 Skills Demonstrated
Data Cleaning & Transformation

Interactive Dashboard Design

Sports Analytics

📂 Data Source
The dataset used in this project was obtained from Kaggle and contains Premier League player statistics, including appearances, goals, yellow cards, red cards, jersey numbers, and team information.

🌍 FIFA World Cup Statistics Dashboard

📖 Overview

The FIFA World Cup Statistics Dashboard is an interactive Power BI project that provides a comprehensive analysis of historical FIFA World Cup tournaments. Built using a dataset sourced from Kaggle, the dashboard enables users to explore tournament trends, attendance figures, goal-scoring statistics, and host locations through dynamic and interactive visualizations.

The project showcases how Power BI can transform historical sports data into meaningful insights, making it easier to identify patterns and compare tournament performance across different years.

🎯 Project Objective

To develop an interactive dashboard that analyses historical FIFA World Cup data, allowing users to explore tournament statistics, attendance trends, scoring patterns, and host locations through intuitive visualizations and dynamic filtering.

📊 Dashboard Features
Interactive Team Filter to view tournament data for individual national teams.
Attendance Analysis showing the distribution of tournament attendance across World Cup editions.
Host City Map displaying geographical locations of World Cup host cities.
Goals by Tournament Year visualized through trend charts to identify scoring patterns over time.
Tournament Information Table containing:
Team Initials
Tournament Stage
Stadium
KPI Cards displaying:
⚽ Total Goals
🎮 Total Matches Played
📈 Average Goals per Match
✨ Key Highlights
Interactive filtering with slicers
Geographic visualization using map charts
Historical trend analysis across World Cup tournaments
KPI cards for quick statistical summaries
Dynamic line and pie charts for comparative analysis
Clean dashboard layout focused on user-friendly exploration
Real-world sports analytics using historical FIFA World Cup data
