#PREMIER LEAGUE STATS

⚽ Premier League Team Statistics Dashboard
An interactive Power BI dashboard built to analyze player and team performance in the Premier League. The dashboard provides a comprehensive overview of key football statistics, enabling users to explore player contributions, disciplinary records, and team-specific insights through dynamic filters and visualizations.

Project Objective: Develop an interactive Power BI dashboard that transforms raw Premier League player statistics into meaningful visual insights, allowing users to analyse player performance, disciplinary records, and team-level metrics through dynamic filtering and intuitive visualisations.

📊 Dashboard Features
Interactive Team Filter to analyse statistics for individual Premier League clubs.
Goal Scoring Analysis displaying the league's top goal scorers.
Yellow Card Distribution to identify the most booked players.
Red Card Distribution highlighting disciplinary trends.
Player Performance Comparison between appearances and goals scored.
Player Details Table containing jersey numbers, player names, and appearances.
Summary KPI Cards displaying:
⚽ Total Goals
🟨 Total Yellow Cards
🟥 Total Red Cards
✨ Key Highlights
Interactive filtering using slicers
Dynamic Power BI visualizations
Clean and modern dashboard layout
KPI cards for quick performance summaries
Comparative player analysis
Drill-down into team-specific statistics
Sports analytics using real-world football data
